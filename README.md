# Rock Paper Scissors!

![Local Image](images/main.PNG)

## Description:
A fun traditional guessing game. 

## Win conditions:

Rock > Scissors<br>
Paper > Rock<br>
Scissors > Paper<br>

Note: The player wins the game by choosing the option before the ">" symbol.

## Keyboard input:
r - Rock<br>
p - Paper<br>
s - Scissors<br>
a - Auto play<br>
Backspace - Reset<br>
